## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Stable internet connection
</Callout>

yangheIDE provides a seamless way to organize your project documentation. Follow these steps to get up and running in under 5 minutes.

## Sign Up and Log In

Create your account and access the dashboard.

<Steps>
  <Step title="Visit yangheIDE" icon="globe">
    Navigate to [https://yangheide.com](https://yangheide.com) and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, create a strong password, and complete the verification.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the login page, enter your credentials, and access your dashboard.
  </Step>
</Steps>

## Create a Documentation Space

Organize projects by creating dedicated spaces.

<Steps>
  <Step title="New Space" icon="plus">
    From the dashboard, click **New Documentation Space**.
  </Step>
  <Step title="Configure" icon="settings">
    Enter a name like `My Project Docs`, add a description, and select visibility (private/public).
  </Step>
  <Step title="Launch" icon="rocket">
    Click **Create** to open your new space.
  </Step>
</Steps>

## Create and Edit Your First Document

Start authoring content immediately.

<Tabs>
  <Tab title="Markdown Mode" icon="code">
    <Steps>
      <Step title="New Document">
        Click **New Document** in your space.
      </Step>
      <Step title="Edit Content">
````markdown
# Welcome to My Project

## Features

- Real-time collaboration
- MDX support
- Custom themes
````
      </Step>
      <Step title="Preview & Save">
        Toggle live preview and click **Publish**.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Visual Editor" icon="type">
    Drag-and-drop blocks for headings, images, and code snippets. No syntax required.
  </Tab>
</Tabs>

<CodeGroup tabs="Markdown,HTML">
```markdown
# yangheIDE Document

## Quick Setup

1. Install dependencies
2. Configure environment
```
```html
<h1>yangheIDE Document</h1>
<h2>Quick Setup</h2>
<ol>
  <li>Install dependencies</li>
  <li>Configure environment</li>
</ol>
```
</CodeGroup>
