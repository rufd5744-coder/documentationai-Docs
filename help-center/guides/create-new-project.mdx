## Start a New Project

<Steps>
  <Step title="Open yangheIDE">
    Launch yangheIDE from your desktop or applications folder.
  </Step>
  <Step title="Choose 'New Project'">
    On the welcome screen, click **New Project**.
  </Step>
  <Step title="Configure project">
    Select a project type (e.g. Node.js, Python), enter a name, and set the location.
  </Step>
  <Step title="Finish">
    Click **Create**. Your new project will open, ready to use!
  </Step>
</Steps>

You're set to start coding in yangheIDE!
